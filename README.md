<h2>概要</h2>
<p>
　このPythonプログラムは、学校の職員室おいて生徒が先生の呼び出しをスムーズに行うために開発しているものです。「TeaChimer」は、このプログラムなどの総称を指します。また、「TeaChimer」の名称の由来は、「teacher」と「chime」を掛け合わせた造語に～する人を表す接尾辞の「er」を付けたことから来ています。そして、開発は日本の学生が行いました。
</p>
